# Project Documentation

## Project Overview

This project is an interactive registration form for the fictional TechConnect 2026 Conference. The application demonstrates HTML form creation, CSS styling, JavaScript validation, accessibility practices, and event handling.

The form validates user input using both built-in HTML validation and custom JavaScript validation with regular expressions.
